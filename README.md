## 2.8. Generics

Generics –≤ TypeScript - —ç—Ç–æ —Å—Ä–µ–¥—Å—Ç–≤–æ –¥–ª—è —Å–æ–∑–¥–∞–Ω–∏—è —É–Ω–∏–≤–µ—Ä—Å–∞–ª—å–Ω—ã—Ö, –ø–µ—Ä–µ–∏—Å–ø–æ–ª—å–∑—É–µ–º—ã—Ö —Ñ—É–Ω–∫—Ü–∏–π, –∫–ª–∞—Å—Å–æ–≤ –∏ –∏–Ω—Ç–µ—Ä—Ñ–µ–π—Å–æ–≤, —Ä–∞–±–æ—Ç–∞—é—â–∏—Ö —Å —Ä–∞–∑–ª–∏—á–Ω—ã–º–∏ —Ç–∏–ø–∞–º–∏ –¥–∞–Ω–Ω—ã—Ö.
–î–∂–µ–Ω–µ—Ä–∏–∫–∏ –ø–æ–∑–≤–æ–ª—è—é—Ç —É–º–µ–Ω—å—à–∏—Ç—å –¥—É–±–ª–∏—Ä–æ–≤–∞–Ω–∏–µ –∫–æ–¥–∞, —Å–ª–µ–¥—É—è –ø—Ä–∏–Ω—Ü–∏–ø—É DRY (Do Not Repeat Yourself).

### –ü—Ä–∏–º–µ—Ä—ã –ò—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è Generics

1. –ë–∞–∑–æ–≤—ã–π –ü—Ä–∏–º–µ—Ä —Å –§—É–Ω–∫—Ü–∏–µ–π:

   –ü—Ä–æ–±–ª–µ–º–∞:
   –ù–µ–æ–±—Ö–æ–¥–∏–º–æ—Å—Ç—å —Å–æ–∑–¥–∞–Ω–∏—è –º–Ω–æ–∂–µ—Å—Ç–≤–µ–Ω–Ω—ã—Ö —Ñ—É–Ω–∫—Ü–∏–π –¥–ª—è —Ä–∞–±–æ—Ç—ã —Å —Ä–∞–∑–Ω—ã–º —Ç–∏–ø–æ–º –¥–∞–Ω–Ω—ã—Ö (–Ω–∞–ø—Ä., —á–∏—Å–ª–∞–º–∏ –∏ —Å—Ç—Ä–æ–∫–∞–º–∏).

   –†–µ—à–µ–Ω–∏–µ:
   –í–≤–µ–¥–µ–Ω–∏–µ —Ç–∏–ø–∞ `T` (–º–æ–∂–µ—Ç –±—ã—Ç—å –ª—é–±—ã–º –Ω–∞–∑–≤–∞–Ω–∏–µ–º), –ø–æ–∑–≤–æ–ª—è—é—â–µ–≥–æ —Ñ—É–Ω–∫—Ü–∏–∏ —Ä–∞–±–æ—Ç–∞—Ç—å —Å –ª—é–±—ã–º –ø–µ—Ä–µ–¥–∞–Ω–Ω—ã–º —Ç–∏–ø–æ–º –¥–∞–Ω–Ω—ã—Ö.

2. –†–∞—Å—à–∏—Ä–µ–Ω–Ω—ã–π –ü—Ä–∏–µ–º:
   –í–æ–∑–º–æ–∂–Ω–æ—Å—Ç—å –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è –Ω–µ—Å–∫–æ–ª—å–∫–∏—Ö –¥–∂–µ–Ω–µ—Ä–∏–∫ –ø–∞—Ä–∞–º–µ—Ç—Ä–æ–≤ (–Ω–∞–ø—Ä., `T` –∏ `K`) –¥–ª—è —Å–æ–∑–¥–∞–Ω–∏—è –±–æ–ª–µ–µ —Å–ª–æ–∂–Ω—ã—Ö —Å—Ç—Ä—É–∫—Ç—É—Ä.
